================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEM: npm install fails
SOLUTION:
  1. Clear npm cache: npm cache clean --force
  2. Delete node_modules: rm -rf node_modules
  3. Delete lock file: rm package-lock.json
  4. Try again: npm install

PROBLEM: Port 5173 already in use
SOLUTION:
  â€¢ Vite will automatically use next available port
  â€¢ Check terminal output for actual port
  â€¢ Or specify port: npm run dev -- --port 3000

PROBLEM: Blank page or errors
SOLUTION:
  1. Open browser DevTools (F12)
  2. Check Console tab for errors
  3. Restart dev server (Ctrl+C, then npm run dev)
  4. Clear browser cache (Ctrl+Shift+Delete)

PROBLEM: Styles not loading
SOLUTION:
  1. Stop dev server (Ctrl+C)
  2. Delete node_modules: rm -rf node_modules
  3. Reinstall: npm install
  4. Start again: npm run dev

PROBLEM: Module not found errors
SOLUTION:
  1. Check file paths are correct
  2. Verify all files exist in src/
  3. Check for typos in import statements
  4. Restart dev server
